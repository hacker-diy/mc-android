# MC / Magic Control Android

Universal Android WebView wrapper for Magic Control devices.

- App name: MC
- Brand text: Magic Control
- Device URL: http://192.168.4.1/
- Package: com.magiccontrol.mc
- WebView file chooser enabled for Lock Screen wallpaper upload.

## Build online with GitHub Actions

Open **Actions** → **Build MC APK** → **Run workflow**.

The debug APK will be uploaded as an artifact named `MC_Magic_Control_debug_APK`.
